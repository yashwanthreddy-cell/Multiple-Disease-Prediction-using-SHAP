This folder is intentionally empty.

To generate the .pkl model files:
  1. Open notebooks/01_diabetes_model_training.ipynb
     → Run all cells → creates saved_models/diabetes_model.pkl

  2. Open notebooks/02_heart_disease_model_training.ipynb
     → Run all cells → creates saved_models/heart_disease_model.pkl

  3. Open notebooks/03_parkinsons_model_training.ipynb
     → Run all cells → creates saved_models/parkinsons_model.pkl

Then run: streamlit run app.py

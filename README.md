# 🩺 HealthAI Explainer — Multi-Disease Prediction + SHAP

## Quick Start

### Step 1 — Install dependencies
```bash
pip install -r requirements.txt
```

### Step 2 — Run notebooks to generate models
Open each notebook in Jupyter / VS Code / Google Colab and run all cells:
```
notebooks/01_diabetes_model_training.ipynb      → saved_models/diabetes_model.pkl
notebooks/02_heart_disease_model_training.ipynb → saved_models/heart_disease_model.pkl
notebooks/03_parkinsons_model_training.ipynb    → saved_models/parkinsons_model.pkl
```

### Step 3 — Launch the app
```bash
streamlit run app.py
```
Open http://localhost:8501

## Project Structure
```
healthai-explainer/
├── app.py                                   ← Streamlit app
├── requirements.txt
├── diabetes.csv
├── heart.csv
├── parkinsons.csv
├── notebooks/
│   ├── 01_diabetes_model_training.ipynb     ← SVM, accuracy ~77%
│   ├── 02_heart_disease_model_training.ipynb← LogReg, accuracy ~80%
│   └── 03_parkinsons_model_training.ipynb   ← SVM, accuracy ~87%

    ├── diabetes_model.pkl      ← generated by notebook 01
    ├── heart_disease_model.pkl ← generated by notebook 02
    └── parkinsons_model.pkl    ← generated by notebook 03
```

## What each notebook shows
- Dataset shape, class distribution charts
- Missing value check
- Correlation heatmap
- Train/Test split
- Model training
- **Accuracy scores (train + test)**
- Classification report (precision, recall, F1)
- 5-Fold Cross Validation scores
- Confusion matrix
- Feature importance / coefficients chart
- Sample prediction verification
- Model saved as `.pkl`
